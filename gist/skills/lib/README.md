# Skills lib

Shared conventions used by multiple skills. These are *reference documents*, not executable skills. Skills cite them via wikilinks or relative paths.

- [`provenance.md`](./provenance.md) — how to attribute durable knowledge to its source.
- [`report-template.md`](./report-template.md) — the compact section shape for check-* sub-skills.

If you find yourself writing a pattern that two or more skills share, add it here and reference it from each skill.
