# Photo Board Sorting Design Chat

A quick voice-note chat; only the coordinator's side is reliable. Keep this as a list of requirements, not a full decision record.

## Extracted Requirements

- [[markers/photo-boards]] should sort each picture into a kind: shelf, tool, sign, map, logo, and related kinds.
- Caption handling should compare any text in the photo to the known shelf labels. The actual text in the picture wins over a general guess.
- Layout labels (`left`, `centered`, `right`) mainly matter for the wide map shots, not every picture.
- Quality/sorting is still being worked out. Start on the workbench and try a few sorting options.
